[English](README.md)

キー操作だけでノートを作り、繋げ、育てるための
**Obsidian向け Zettelkasten プラグイン**。

UI操作に頼らず、思考の流れを止めないことを最優先に設計しています。

---

## コンセプト

- キー操作だけで完結
- **モード**でノートの種類（Core / Temp / Ref など）を切り替える
- 操作は UI ではなく **文脈（モード・選択テキスト・カーソル位置）** で決まる

---

## できること

| コマンド | 説明 |
|---|---|
| ZKey: Zettelを開く・作る | テキスト選択中は即リンク化＋作成、カーソルがリンク上なら開く、それ以外はノート名を入力して作成 |
| ZKey: リンクスイッチャー | 現在のノートのリンク・バックリンク・2stepリンクを一覧表示して開く |
| ZKey: rootノートに戻る | 現在のモードの起点ノートを開く |
| ZKey: モード切り替え | モード一覧から切り替える |
| ZKey: モードを作成 | モードを新規作成する（フォルダ・rootノート・テンプレートを自動生成） |
| ZKey: モードを削除 | モード一覧から削除する |

---

## モードとは

**モード**は「ノートの種類ごとの作業領域」です。Core（本論）・Temp（一時メモ）・Ref（参考資料）のように用途別にモードを作成し、切り替えながら使います。

各モードには以下が紐付きます：
- **フォルダ**：ノートの保存先
- **rootノート**：モードの起点となるノート
- **テンプレート**：新規ノート作成時に使うテンプレートファイル

---

## 設定

Obsidianの設定 → ZKey から以下を変更できます。

| 設定 | 説明 | デフォルト |
|---|---|---|
| モードの自動切り替え | ファイルを開いたとき、そのファイルのモードに自動で切り替えます | オン |
| 本文へのzk-origin挿入 | モード作成時のテンプレートに `↑: [[{{zk-origin}}]]` を自動追加します | オフ |
| デフォルトノートフォルダ | モード作成時のフォルダパスの初期値 | `Zk` |
| テンプレートフォルダ | Obsidian Templates プラグインの設定から自動取得されます | – |

## インストール

**Obsidian コミュニティプラグイン（近日対応予定）**

設定 → コミュニティプラグイン → 検索 → `ZKey`

**BRAT（ベータ版）**

1. [BRAT](https://github.com/TfTHacker/obsidian42-brat) をインストール・有効化
2. BRATの設定から「Add Beta plugin」を選択
3. このリポジトリのURLを入力
