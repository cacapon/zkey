**2. Along with the mode creation, [[{{temp-name}}]] has been created.**

Special keywords:
- `{{zkid}}` — a unique Zettel ID is automatically assigned
- `{{alias}}` — a shortened unique alias derived from zkid is inserted
- `{{zk-origin}}` — a link to the note that was open when the Zettel was created is automatically inserted
